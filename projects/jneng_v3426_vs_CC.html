三、根因诊断

为什么 V3 缺失了这些内容？

根因 1：V3 是"快速生成"版本，不是"深度尽调"版本

V3 业务报告来源：

1. 复跑 V2 业务报告（V3.42.4 学术研究增强版）
2. 用了相同的 MD 源（jneng_v3424_v2.md）
3. 只升级了 CSS 视觉效果（KPI 4 色 + Timeline className）
4. 没有补内容深度

根因 2：V3 报告严重依赖"本地收资材料"，缺少"行业分析/可比公司/估值方法"原始数据

V3 数据来源分布（来自 V3 报告附录 8.1 MANIFEST）：

CC 的数据广度是 V3 的 5 倍以上。

根因 3：V3 没有"投资亮点 / 投资建议 / 最终判断"——这是尽调报告的灵魂

CC 第五部分（综合结论）： - 5.1 投资亮点：6 大亮点（技术护城河/客户结构/盈利断层/IDM/战略协同/管理层） - 5.2 风险提示：6 大风险（MBO 包袱/应收账款/客户集中/行业周期/换股方案/政府补贴） - 5.3 投资建议：7 维度评级 - 5.4 最终判断：800 字核心论断

V3 第七部分（风险总览）： - 🔴 红色风险（3 项） - 🟡 黄色风险（3 项） - 🟢 绿色信号（3 项）

V3 只有"风险"和"信号"，没有"投资亮点"、"投资建议"、"最终判断"——这是最严重的差距。

五、给主公的关键判断

主公当前问的"如何保持内容完整性"，根本答案是：

CSS 视觉升级 ≠ 内容升级。

V3.42.6 修复的是 CSS 视觉（KPI 4 色 / Timeline 显式 className），不是 业务内容深度。

CC 原版的"行业分析 / 投资亮点 / 最终判断 / 数据置信度"——这些是 数据 + 推理 + 结论 的深度内容，不可能靠 CSS 升级出来，必须由 data 代理 + research 代理 + main 代理共同完成。
